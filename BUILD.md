## 🔑 게이트 (너가 채울 것 — 빌드 전 준비)

> ⚠️ API 키·패스코드는 **이 파일이나 채팅에 적지 마라.** 빌드 때 `.env`(gitignore됨) 자리가 생긴다. 지금은 *발급/결정만* 해둘 것.

### 1. Anthropic API 키
[console.anthropic.com](https://console.anthropic.com) → API Keys에서 발급. 네가 보관. 빌드 후 로컬 `.env`의 `ANTHROPIC_API_KEY`에 입력.

### 2. 패스코드 (PWA 로그인용)
원하는 비밀 문자열 하나 정해둠. 빌드 후 `.env`의 `APP_PASSCODE`에 입력.

### 3. ~~관심사 프로필~~ → **불필요** (콜드스타트 설계)
산문 안 쓴다. v1은 화제도로 시작하고, 읽기 기록 대시보드가 *진짜 관심사를 드러낸다.* 그 데이터로 나중에 채운다. **지금 할 일 없음.**

### 4. 소스 (비우면 기본값 — 나중에 PWA에서 수정)
- 기본: Hacker News(API), Reddit r/programming·r/MachineLearning, Anthropic 블로그 RSS, arXiv cs.AI·cs.SE, (선택) 팔로우 GitHub repo 릴리스
- 추가/제외할 것 있으면 새 세션에서 말로 전달.

## 📦 빌드 스코프 (내가 한 스윕에 — 너 없이 가능한 전부)

**로컬에서 end-to-end로 도는 v1 전체:**

1. **프로젝트 골격** — 단일 repo에 백엔드(Hono)+프론트(SvelteKit), TS 설정, `.gitignore`(.env 포함), `.env.example`
2. **SQLite 스키마** — config / sources / items(+genre) / seen / briefings / push_subs / **read_events**  (interests 테이블은 v2 예약)
3. **수집 계층** — RSS/Atom 통합 fetcher + HN·Reddit API 어댑터
4. **선별 파이프라인 (콜드스타트)** —
   수집 → [코드] 이미본것 제거·교차소스 중복병합(중복=화제도 강신호)·오래된것 컷 → 소스별 **화제도 정규화**(z-점수)
5. **AI 계층** — Haiku 4.5로 각 기사 **8장르 분류**(적합도 점수 ❌) → Sonnet 4.6으로 상위~10건 요약
   - 8장르: AI/LLM·에이전트 / 시스템·인프라·저수준 / 보안 / 논문·연구 / 언어·런타임·도구 / 웹·프론트 / 제품·창업·회고 / 기타
6. **랭킹** — **필독 3 = 화제도 + 다양성 가드**(같은 장르 페널티) / **더보기 = 전 장르 고루 샘플**(저화제 논문·블로그 *탐색 주입* → 학습용)
7. **브리핑 생성** — 레이어드(필독3 문단 + 더보기 한줄), DB 저장
8. **읽기 기록 + 대시보드** — 어떤 링크 열었나 `read_events`에 로깅 → **토픽(장르)별 클릭 집계 대시보드** (자동학습은 v2)
9. **스케줄러** — node-cron. config는 **도착 시간**(기본 05:00). 파이프라인은 도착 N분 전(기본 04:45)에 수집·AI를 끝내고, 푸시는 05:00 정각 발송 → 5시에 *완성된 보고서가 도착*
10. **API** — Hono 라우트(브리핑 조회, 읽기 이벤트 기록, 대시보드 집계, 설정 get·set, 푸시 구독) + 패스코드/JWT 인증
11. **PWA** — SvelteKit: 로그인 → 브리핑 뷰(레이어드) → 대시보드(장르별 클릭) → 설정(도착시간·소스) → 설치 매니페스트 + 서비스워커 + 푸시 구독 플로우
12. **web-push 배선 + VAPID 키 생성 스크립트** (VAPID는 코드로 생성)
13. **README** — 로컬 실행법

**검증:** 빌드 후 `/verify`·`/run`으로 실제 구동 확인(말로 "됐다" 금지).

## ⚙️ 기본값(나중에 튜닝)
- 콜드스타트 랭킹 = 화제도 100% (적합도 가중치는 v2에서 데이터 기반 부활)
- 더보기 개수 N = 7, 그 중 탐색 주입(저화제 전장르 샘플) 비중 ≈ 절반

- 도착 N분 전 = 15분
